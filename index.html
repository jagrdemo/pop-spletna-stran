Priprava spletne strani

Za pripravo spletne strani je dobro poznati osnove

  • jezika HTML,
  • jezika CSS,
  • jezika JavaScript.

Gradiva in vodičev je na spletu ogromno. Za hitro ponovitev ali uvod si lahko na primer ogledate posnetke v YouTube kanalu Web Demystified.

Pri pripravi datotek si lahko pomagamo z urejevalnikom Visual Studio Code. Za predogled nastajajoče spletne strani uporabimo paket HTML Preview.

Objava spletne strani

Spletno stran lahko objavimo s pomočjo različnih ponudnikov spletnega gostovanja. Eden takih je Netlify, ki omogoča, da spletno stran naložimo tudi preko javnega Git repozitorija, na primer GitHub.

Spletna stran je bila pripravljena v okviru predavanj Programska oprema pri pouku (FMF, UNI-LJ) aprila 2021.
